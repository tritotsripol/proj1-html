Input Type

<label for="(Variable)">(Title):</label>
<input type="(Variable)" id="(Variable)" name="(Name)" >

Example Output:

    • Other Attributes Meaning
    value="(Variable)" Set (Variable) to default
    min="(Variable)" Set (Variable) to minimum
    max="(Variable)" Set (Variable) to maximum
    step="(Variable)" Adjust in (Variable) steps
    placeholder="(Text)" Shows (Text) in a form box before enters input.

    Type of Min or Max to Date Variables

    Date Variables Min or Max
    date YYYY-MM-DD
    month YYYY-MM
    week YYYY-WW
    time HH:MM:SS
    datetime YYYY-MM-DDTHH:MM:SS
    datetime-local YYYY-MM-DDTHH:MM:SS

    Options

    <input type="(checkbox or radio)" id="option(no.)" name="option" />
    <label for="option(no.)">(Text)</label>
    ...(depends on the number of options)

    Multiple option: "checkbox" Single option: "radio"
    Example Output
    Set to defult <input type="checkbox" id="option(no.)" name="option" checked/> <input type="checkbox" id="option(no.)" name="option" value="(Variable)"/>

    <select id="option" name="option">
    <option value="option(no.)">(Text)</option>
    ...(depends on the number of options)
    </select>

    or

    <select id="option" name="option">
    <optgroup label="(Group name)">
    <option value="option(no.)">(Text)</option>
    ...(depends on the number of options)
    </optgroup>
    ...(depends on the number of optgroups)
    </select>

    Select box: <option> Select box(group): <optgroup>
